Category: File Management
Definition: for displaying a list of information about files

Options:

  • -a, --all
    • doesn't ignore entries with "." as its start (default)
  • -A, --almost-all
    • do not list "." and ".."
  • -author
    • requires -1, will print author of each file
  • -b, --escape
    • print C-style escapes
  • -block-size=SIZE
    • scale sizes by SIZE specificed
  • -B, --ignore-backups
    • ignores entires with ~
  • -c
    • use with -lt: will sort and show by ctime
    • use with -l: show ctime and sort by name
  • -C
    • list entries in/by columns
  • --color[=WHEN]
    • gives the output color
