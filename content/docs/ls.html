Category: File Management
Definition: for displaying a list of information about files

Options Explanation
-a, --all doesn't ignore entries with "." as its start
this is the default
-A, --almost-all do not list "." and ".."
--author requires -1, will print author of each file
-b, --escape print C-style escapes
--block-size=SIZE scale sizes by SIZE specified
-B, --ignore-backups ignores entries with ~
-c use with -lt: will sort and show by ctime
use with -l: show ctime and sort by name
-C list entries in/by columns
--color[WHEN] gives the output color
WHEN can be always, never, or auto
-d, --directory list directory entries instead of the directory files
does not dereference symlinks
-D, --directd generate output for EMACS
-f says do not sort
-F, --classify ppend an indicator to entries
indicators: */=>@|
--file-type similar to --classify, but cannot append *
--format=WORD formats according to the WORD
Options: -x, -m, -X, -l, -1, -C
--full-time same as --time-style=full iso
-g like -l but does not list owner
--group-directories-first lists the group directories before files
--sort=none will disable this command
--time, --time=WORD shows time of last modification of any file in the directory
WORD means it'l show a word instead such as: atime, access, use, ctime, or status
-G, --no-group in long listings, the long group names aren't printed
-h, --human-readable
--si		 use with -l
prints sizes in human readable formats
--si prints in powers of 1000 not 1024
--hide=PATTERN hides all entries matching the shell PATTERN specified
--indicator-style=WORD append indicator to each entry
-i, --inode prints index number of each file
-I, --ignore=PATTERN ignored the entries that match the specified PATTERN
-k, --kibibytes uses 1024 blocks
-l uses a long listing format
-L, --dereference shows information for the file the link references rather than the symlink
-m full width with a comma separated list of entries
-n, --numeric-uid-grid lists numeric user and group ids
like -l
-N, --literal print raw entry names
-o lists everything but group information
like -l
-p, --indicator-style=slash append / indicator to directories
-q, --hide-control-chars print ? instead of non graphic chars
--show-control-chars shows non graphic chars as-is
-Q, --quote-name entries names are enclosed in quotes
-r, --reverse reverse order while sorting
-R, --recursive lists subdirectories recursively
-s, --size print allocated size of each file
-S sorts files by size
--sort=WORD sorts by the WORD instead of name
WORD: none/-U, extension/-X, size/-S, time/-t, version/-v
-T, --tablesize=COLS assume tab stops at each COLS instead of 8
-U doesn't sort, list in order
-w, --width=COLS assumes screen width of COLS
-x list entries by lines instead of columns
-X sort by extension, alphabetically
-Z, --context print any SELinux security context for each file
-1 list one file per line

Examples:

ls -l
ls -laxo
ls /
ls -ltr
