Category: File Management
Definition: to move or remname files.

Options:

  • -b, --backup[=CONTROL]
    • make backup of each file
    • -b doesn't accept an argument but --backup does
  • -f, --force
    • prevents a prompt from appearing before overwriting a file
  • -i, --interactive
    • prompt before overwriting each destination file
    • disregards file premissions
  • -n, --no-clobber
    • do not overwrite an existing file
      • ,/ul>

    Examples:

    • mv file1.txt Sites
      • moves the file file1 to the Sites directory (destination)
    • mv file1.txt file3.txt
      • renames the file1.xt as file3.txt
    • mv file1.html ../
      • moves file1 up to the parent directory
