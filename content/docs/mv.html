Category: File Management
Definition: to move or rename files.

Options Explanation
-b, --backup[=CONTROL] make backup of each file
-b doesn't accept an argument but --backup does
-f, --force prevents a prompt from appearing before overwriting a file
-i, --interactive prompt before overwriting each destination file
disregards file permissions
-n, --no-clobber do not overwrite an existing file

Examples:

mv file1.txt Sites
mv file1.txt file3.txt
mv file1.html ../
