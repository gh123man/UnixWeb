Category: File Management
Definition: to remove files or directories. In order to remove a directory, the -r option must be specified. Otherwise the rm command will only remove files.

Options Explanation
-f, --force ignores files that are non-existent
does not prompt before removing
-i gives a prompt before each removal
-I prompts only once before removing if there are 3+ files being removed
prompts only once if removing reclusively
--interactive[=WHEN] prompt removes as specified by WHEN
WHEN can be: always, once, never
default for WHEN is always
--one-file-system sfor when you are removing a hierarchy recursively
says skip directories that is on a different file system than on the cmd line
--no-preserve-root / is not treated in a special way
/ is the root directory
--preserve-root will not remove the root directory aka "/"
this is the default setting
-r, -R, --recursive removes directories and their contents recursively
-v, --verbose explains what is happening for each action
--help display help message and exits
--version displays version and exits

Examples:

rm file1.txt
rm -f *
rm -r Sites
rm -fr Sites
rm -I *.txt
