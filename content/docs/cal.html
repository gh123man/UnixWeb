Category: System Info
Definition: for displaying a calendar on the command line. The calendar is formatted for you after command is called

Options Explanation
-l displays a single month
this is the default
-3 displays three months: previous month, the current month, and the upcoming month
-s display the calendar with Sunday as the starting day
-m displays the calendar with Monday as the starting day
-j displays the calendar with dates of the Julian calendar
-y display a calendar for the entire current year

Examples:

cal
cal 12 2000
cal -j
cal -y
