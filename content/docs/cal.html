Category: System Info
Definition: for displaying a calendar on the commanxd line. The calendar is formatted for you after called

Options:

  • -l
    • displays a single month (default)
  • -3
    • displays three months: previous month, the current month, and the upcoming month
  • -s
    • display the calendar with Sunday as the starting day
  • -m
    • displays the calendat with Monday as the starting day
  • -j
    • displays the calendat with dates of the Julian calendar
  • -y
    • display a calendar for the entire current year

    • Uses

  • cal
    • display the calendar for the current month
  • cal month year
    • displays the calender for the month and year specificed/li>

    • Examples:

    • cal 12 2000
    • cal -j
    • cal -y
