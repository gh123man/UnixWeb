Category: System Info
Definition: displays the date and time as well as setting the date and time. Please note you must be the root user to set the time

Options:

  • -a / --archive
    • perserved
  • --attributes only
    • refers to the previous job
  • %(num)
    • refers to the job number
    • i.e. %1 or %2

    • Uses:

    • date (time)
      • sets the time indicated by (time)
    • date (date/time)
      • sets the date and time indicated by (date/time)

      • Uses:

      • date
      • date 0530.30
      • date 10250045
      • date --set="20091015 04:30"
