Category: System Info
Definition: shows the amount of disk space available and the amount of disk space being used for file systems.

Options Explanation
-a, --all includes dummy file systems
-B, --block-size=SIZE scales the file by size before printing
--total displays the total size; a grand total
-h, --human-readable
-H, --si		 print size in a 'human readable' format i.e. 1K 234M
-H & --si use powers of 1000 instead 1024
-i, --inodes list inode information
-k same as --block-size=1K
-l, --local only displays local file systems
--no-sync prevents a sync from happening before getting information
this is the default
-P, --portability makes it so the format is in POSIX
--sync implements a sync before getting information
-t, --type=TYPE limit the listing to a certain TYPE of file
-T, --print-type prints the file system type with other information
-x, --exclude-type=TYPE excludes the TYPE of file system specified

Examples:

df
df Sites
df Sites -h
df --local
df --sync
