Category: File Management
Definition: for copying files or directories.

Options Explanation
-a, --archive archives the file/directory
--attributes-only says: don't copy the data, just create a file with the same attributes as the file to be copied
-b, --backup[=CONTROL] make a backup of existing file
-b doesn't accept a control argument (uses default) but --backup does
=CONTROL specifies the version control method to use
--copy-contents When operating recursively, the command instructs to copy contents of special files
-f, --force if an existing destination file cannot be opened, remove file & try again
if -n is used, the -f command will not have an effect
-i, --interactive prompt before overwrite
will override -n option
-H follow command line symlinks
-l, --link creates hard links instead of copying
-L, --dereference always follow symlinks in SOURCE
if the SOURCE is a symlink: copy file linked, not the symlink
-n, --no-clobber don't overwrite an existing file
overwrites the -l option
-P, --no-dereference never follow symlinks in the source
-p, --preserve[=ATTR_LIST] preserves the specified attributes
-p is a default the same as --perserve=mode,owenrship,timestamps
attributes: mode, ownership, timestamps, context, links, xattr, all
--parents use full source file name
-R, -r, --recursive copy files recursively
--reflink[=WHEN] control clone copies
--remove-destination removes every destination file before opening
opposite of --force
--sparse=WHEN control creation of sparse files
-s, --symbolic-link makes a symbolic link instead of copying the file/directory
-S --suffix=SUFFIX overrides the default backup suffix
-t, --target-directory=DIRECTORY copy all source arguments into the DIRECTORY specified
-T, --no-target-directory treats as a normal file
-u, --update copy only when the SOURCE file is newer than the destination file
-v, --verbose prints the details and explanations of the actions occuring
-x, --one-file-system stay on this file system

Examples:

cp file1.txt myDir
cp /home/Sites/file.txt /home/myDir/file.bak
cp *.html /home/Sites/
cp -u *.html Sites
