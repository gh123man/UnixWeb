Category: System Info
Definition: shows the amount of disk sapce abailable and the amount of disk space being used for files.

Options:

  • -a, --all
    • includes dummy file systems
  • -B, --block-size=SIZE
    • scales the file by size before printing
  • -b, --bytes
    • the same as --apparent-size --block-size=1
  • -c, --total
    • displays the total size; a grand total
  • -D, --dereference-args
    • dereferences only symlinks that are listed by you
  • --files0-from=F
    • summarize disk ussuage of NUL-terminated file names as specified in File F
  • -h, --human-readable, --si
    • print size in a 'human readable' format
    • i.e. 1K 234M
    • -H & --si use powers of 1000 instead 1024
  • -k
    • same as --block-size=1K
  • -l, --count-links
    • count sizes many times if hard linked
  • -m
    • like --block-size=1M
  • -L, --dereference
    • dereference all symlinks
  • -P, --no-dereference
    • don't follow any symlinks
    • this is the default
  • -0, --null
    • end each ouptut line with 0 byte
    • default is a newline
  • -s, --separate-dirs
    • do not include subdirectories in list
  • -s, --summarize
    • display only a total for each item
  • -x, --one-file-system
    • slip direcotries not in local file system
  • -X, --exclude-from=FILE
    • exclude files that match any pattern in FILE
  • --exclud=PATTERN
    • exclude any files that match the PATTERN specififed
  • --time, --time=WORD
    • shows time of last modification of any file in the directory
    • WORD means it'l show a word instead such as: atime, access, use, ctime, or status

Uses/Examples:

  • du -s *.txt
    • shows a summary of the size of every file that's a .txt file
  • du -shc *.html
    • shows a summary of the size in readable format of every .html file
  • df Sites -h
    • displays the same as above, but in a readable format
