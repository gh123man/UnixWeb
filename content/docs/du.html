Category: System Info
Definition: shows the amount of disk space available and the amount of disk space being used for files.

Options Explanation
-a, --all includes dummy file systems
-B, --block-size=SIZE scales the file by size before printing
-b, --bytes the same as --apparent-size --block-size=1
-c, --total displays the total size; a grand total
-D, --dereference-args dereferences only symlinks that are listed by you
--files0-from=F summarize disk usage of NULL-terminated file names as specified in File F
-h, --human-readable
--si		 print size in a 'human readable' format i.e. 1K 234M
--si use powers of 1000 instead 1024
-k same as --block-size=1K
-l, --count--link count sizes many times if hard linked
-m like --block-size=1M
-L, --dereference dereference all symlinks
-P, --no-dereference never follow symlinks in the source
this is the default
-0, --null end each output line with 0 byte
default is a newline
-s, --separate-dirs do not include sub-directories in list
-s, --summarize display only a total for each item
-x, --one-file-system skip directories not in local file system
-X, --exclude-from=FILE exclude files that match any pattern in FILE
--exclude=PATTERN exclude any files that match the PATTERN specified
--time, --time=WORD shows time of last modification of any file in the directory
WORD means it'l show a word instead such as: atime, access, use, ctime, or status

Examples:

du -s *.txt
du -shc *.html
df Sites -h
