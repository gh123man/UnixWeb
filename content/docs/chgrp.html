Category: File Management
Definition: changes a group ownership of a file or of many files.

Options Explanation
-C, --changes reports when a change is made
-f, --silent, --quiet makes it so error messages are not shown
-v, --verbose prints out details/diagnostic for every file processed/changed
-h, --no-dereference affects symbolic links instead of any referenced file
can only use this if you can change symlink ownership
--no-preserve-root do not treat '/' in a special way
this is the default setting
--reference=FILENAME use FILENAME's group settings rather than specifying a group value
-R, --recursive operates recursively on files and directories
these options modify how hierarchy is affected when -R is specified:
   -H: traverses a symlink if it's on the cmd line
   -L: traverses every symbolic link to a directory
   -P: do not traverse any symbolic links (default)
--help cdisplay help message and exits
--version displays version and exits

Examples:

chgrp users file.txt
chgrp -hR admin /database/
chgrp --reference=STAFF finances.txt
